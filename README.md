## 🚀 Project Summary
**Business Insights 360** is an end-to-end Power BI analytics project built for **AtliQ Hardware** to enable data-driven decision-making across key business domains.

The goal was to transform raw data into actionable insights across **Finance, Sales, Marketing, and Supply Chain**, helping stakeholders monitor performance, identify risks, and drive strategic growth.

🔗 **Live Dashboard:** *[Power BI Dashboard]* (https://app.powerbi.com/view?r=eyJrIjoiN2M4ZTVlMzUtOGM1YS00M2RmLWFkYmEtNTNkYzc3Yjk1MzdmIiwidCI6ImM2ZTU0OWIzLTVmNDUtNDAzMi1hYWU5LWQ0MjQ0ZGM1YjJjNCJ9)

## 🎯 Business Problem
AtliQ Hardware expanded rapidly but faced losses due to decisions driven by intuition and limited analytics.

**Challenges:**
- Lack of centralized analytics system  
- Limited visibility into business performance  
- Inefficient decision-making processes  

**Solution:**  
Develop an interactive Power BI dashboard to provide real-time, data-driven insights across all departments.

---

## 💡 Key Features

- 📊 Multi-page interactive dashboard (Finance, Sales, Marketing, Supply Chain)  
- 🔄 Dynamic filtering and navigation using bookmarks & buttons  
- 📈 KPI indicators for performance tracking  
- 🎯 Dynamic titles based on user selections  
- ⚡ Optimized performance using DAX Studio  
- 🎨 Conditional formatting for better data storytelling

## 🏗️ Project Workflow

1. **Requirement Gathering**
   - Defined business objectives and stakeholder expectations  

2. **Data Collection**
   - Imported data from MySQL into Power BI  

3. **Data Transformation**
   - Cleaned and prepared data using Power Query  

4. **Data Modeling**
   - Built relationships and optimized model structure  

5. **Dashboard Development**
   - Designed interactive and user-friendly reports  

6. **Deployment**
   - Published to Power BI Service with scheduled refresh
